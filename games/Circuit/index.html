Circuit

Project Description

Using a simple set of two controls, the player must solve geometric puzzles in space. Objects in your environment can be interacted with, either rotated or shifted to a new location, in order to create a "circuit" that will guide an energy sphere to its destination.

  • First Person Puzzle Game
  • 5 Week Project
  • Primary Role: Lead Programming/Music
