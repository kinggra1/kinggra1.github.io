Sausage Transporter

Project Description

Designed as an exercise in game expandable game design, Sausage Transporter is a dungeon crawler engine written in raw JS. All player interaction, as well easy systems for creating room and map layouts were designed and implemented from scratch.

  • Modular Dungeon Cralwer
  • Solo Endeavor in Game Systems Design
  • Demo of Programming/"Art"/Music
